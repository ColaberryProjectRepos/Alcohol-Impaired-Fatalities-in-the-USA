Step 1: Create the measures

  1. Select the table "DUI_Fatalities "  
  2. Click the new measure button on the ribbon.
  3. Copy and paste the Dax below one at a time and then click the check mark, or Click “enter”


Measures:

  •   Total_Fatalties = COUNTROWS(FILTER(DUI_Fatalities, DUI_Fatalities[Alcohol_Detect]= "Yes (Alcohol involved)"))
  •  Total Injuries = COUNTROWS(FILTER(DUI_Fatalities,DUI_Fatalities[Alcohol_Injureies]="Fatal Injury (K)"))
  •  AccidentPerDay = AVERAGEX ( VALUES (‘Calendar'[Date] ), DUI_Fatalities[Total Injuries])

          v   Change to (Whole Number)

 

  •   %Fatalities = DIVIDE([.Total_Fatalties],[.Total Injuries])

v   Change to (Percentage, 0 decimal Point)

 

  •  YOY =    VAR __PREV_YEAR = CALCULATE(DUI_Fatalities[%Fatalities],DATEADD('Calendar'[Date],-1, YEAR))

                         RETURN

                                DIVIDE(DUI_Fatalities[%Fatalities] - __PREV_YEAR, __PREV_YEAR)

                       

v  Change to (Percentage,  1  decimal Point)

 

  •   Arrow YOY = SWITCH(TRUE(), [YOY] < 0, UNICHAR(9660), UNICHAR(9650))

  •   ColorArrow YOY = IF ([Arrow YOY] = UNICHAR (9660), "Green", "Red")
