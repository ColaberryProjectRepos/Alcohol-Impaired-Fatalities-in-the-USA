Create Calendar table 8

 Step 1: Create Calendar Table

1.  Select the Modeling tab

2.  Click New Table 

3.  Enter the DAX Expression listed above or download the Dax expression from the Red Icon  and Press "Enter".


Calendar table Dax

Calendar =

ADDCOLUMNS(

           CALENDAR(  MIN(DUI_Fatalities[Date]),

                      MAX(DUI_Fatalities[Date])

           ),

        "Year", YEAR([Date]),

        "Quarter Number", INT( FORMAT([Date], "q") ),

        "Quarter","Q" & INT(FORMAT( [Date], "q") ),

        "Month Number", MONTH( [Date]),

        "Month", FORMAT( [Date], "mmm" ),

        "Week Day Number", WEEKDAY ( [Date] ),

        "Week Day", FORMAT ( [Date], "dddd" ),

        "Year Month Number", YEAR ( [Date] ) *100 + MONTH ( [Date] ),

        "Year Month", FORMAT ( [Date], "mmm" ) & " " & YEAR ( [Date] ),

        "Year Quarter Number", YEAR ( [Date] ) * 100 + INT ( FORMAT ( [Date], "q") ),

        "Year Quarter",  "Q" & FORMAT ( [Date], "q") & "-" & YEAR ( [Date] ))

Step 2: Create Relationship between DUI_Fatalities Table and Calendar Table and US_Holiday_Date Table

2.      Drag and drop DATE column from DUI_ Fatalities table to Date column in Calendar Table.
